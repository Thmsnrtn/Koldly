# Koldly Design System

## Color Palette
- Primary: #FF6B35 (accent)
- Dark BG: #0A0A0A
- Card BG: #141414
- Hover: #1A1A1A
- Text: #F5F5F5
- Text Muted: #888
- Border: #222
- Input BG: #0F0F0F

## Typography
- Display: Space Grotesk 700 (headings)
- Body: DM Sans 400-600 (content)

## Component Standards
- Border radius: 8-12px
- Padding: 16px/24px
- Transitions: all 0.2s ease
- Loading: spinner animation
- Error: red accent + message
- Empty: center icon + CTA
