# Koldly

AI-powered sales development that finds your ideal prospects, researches their pain points, and writes personalized cold emails at scale.

**Live:** https://koldly.com

## Stack

- Express.js + PostgreSQL (Neon)
- OpenAI GPT-4o-mini for prospect research and email generation
- Vanilla JavaScript frontend

## Features

- **Smart Prospecting**: AI finds 20-25 perfect-fit prospects matching your ICP
- **Personalized Emails**: Custom-written emails based on prospect research
- **Campaign Management**: Full CRUD for campaigns, prospects, and email sequences
- **Analytics**: Real-time tracking of opens, clicks, replies, and engagement
- **Integrations**: Slack notifications and webhooks for external services

## Deployment

Deployed automatically on Render via `render.yaml`. Every git push triggers a new deployment.
